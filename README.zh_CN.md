# egg-dingtalk-isv

钉钉ISV应用Egg.js插件。

## 安装插件

```bash
npm i @axolo/egg-dingtalk-isv --save
```

## 开启插件

```js
// config/plugin.js
exports.dingtalkIsv = {
  enable: true,
  package: '@axolo/egg-dingtalk-isv',
};
```

## 详细配置

请到 [config/config.default.js](config/config.default.js) 查看详细配置项说明。
